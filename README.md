## 🚀 Proyectos

### 🔐 [Guardian Graph System](https://github.com/srwangcr/Guardian_Graph_System)
Sistema de defensa activa para entornos Linux. Genera honeypots, asigna niveles de riesgo por usuario y activa medidas de contención (Docker isolation, monitoreo de red) cuando detecta cifrado sospechoso en archivos señuelo.  
`Python` · `Cybersecurity` · `Modular Architecture` · `GPLv3`

### 📦 [Nexus Gestión](https://github.com/srwangcr/nexus-gestion)
Sistema full-stack de gestión de órdenes y proveedores con diseño minimalista dark. Dashboard con estadísticas en tiempo real, CRUD completo, autenticación JWT y rutas protegidas.  
`React` · `Node.js` · `Express` · `PostgreSQL` · `JWT`

### 🌐 [Nexus Gateway](https://github.com/srwangcr/nexus-gateway)
API Gateway con rate limiting (Token Bucket & Sliding Window), autenticación JWT, reverse proxy y almacenamiento en Redis. Arquitectura en capas con TypeScript.  
`TypeScript` · `Node.js` · `Redis` · `Docker`

### 🤝 [Ayuda Tica MVP](https://github.com/srwangcr/mvp_7600)
Plataforma web para conectar personas con donantes y reportar violaciones a la Ley 7600 de accesibilidad en Costa Rica. Mapa interactivo de casos, panel multi-rol, integración con Sinpe Móvil.  
`React` · `Node.js` · `PostgreSQL` · `Nginx` · `Cloudflare Tunnel`

### 🧬 [EvolucionIA — Economic Simulator](https://github.com/srwangcr/Ia_Economy_Sim)
Simulador de mercado con agentes heterogéneos (mineros, especuladores, consumidores), algoritmos genéticos con DEAP, lógica difusa para toma de decisiones y dashboard en Streamlit con modo de reproducción tick a tick.  
`Python` · `DEAP` · `Streamlit` · `PostgreSQL` · `Alembic`

### 👻 [KELPIE-GHOST](https://github.com/srwangcr/-KELPIE-GHOST)
Herramienta de pentesting que crea interfaces de red virtuales efímeras con MACs e IPs aleatorias para análisis de conectividad y evasión de detección básica.  
`Python` · `Scapy` · `Networking` · `Pentesting`
